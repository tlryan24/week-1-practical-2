Not sure where to start? Consult the examples shown in lecture and click the Hint button below.

You only need to edit sketch.js.

  1. You will need to write both setup() and draw()functions.
  2. Call createCanvas() in setup() to create the canvas.
  3. The finished sketch should have four squares but you do not need to draw all of them. You can use the background colour of the sketch to create two of the squares.
  4. The squares have no stroke.
  5. The shade of grey used for the bottom right square is 150.
