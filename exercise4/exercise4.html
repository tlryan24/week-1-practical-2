Guidance

This is an open-ended exercise so there is no expected output. Be creative!

Copy and paste your code from sketch.js in any of the stage 2 or 3 exercises into exercise4/sketch.js. Edit the code to add RGB or RGBA stroke and fill to the shapes. You might also like to experiment with changing the thickness of the stroke around your shapes by using the strokeWeight() function.

The image below shows an example, created by starting with exercise 3.1 code. The stroke weight was set to 20, an RGB colour was used for the background, and RGBA colours were used for the stroke and fill. Your sketch doesn't have to match this output!

You only need to edit sketch.js.
